# はじめに

## インストール

[Releases](https://github.com/ak110/lc/releases)からZIPをダウンロードし、任意のフォルダに展開する。
`らんちゃ.exe` を起動するとタスクトレイに常駐し、設定したホットキーでランチャーを開ける。

## 初回セットアップ

1. `らんちゃ.exe` を起動する
2. タスクトレイアイコンを右クリックして「設定」を開き、ホットキーを設定する
3. コマンドを登録する（設定画面から追加、またはファイルをドラッグ&ドロップ）
4. Windowsの「送る」メニューにショートカットを登録すると、エクスプローラーから直接コマンドを追加できる。コマンド型ランチャーの「送るへコマンド登録用ショートカットを作成する」ボタンを使う

## コマンドライン引数

| 引数       | 動作                                                                                |
| ---------- | ----------------------------------------------------------------------------------- |
| `/close`   | 既に常駐している場合、終了する                                                      |
| `/restart` | 既に常駐している場合、再起動する                                                    |
| それ以外   | ファイル名と見なし、コマンドを登録する（Windowsの「送る」メニューからの登録と同等） |
