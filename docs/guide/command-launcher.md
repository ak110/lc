# コマンド型ランチャー

コマンドの一覧を表示し、入力に応じて絞り込んで実行するランチャーである。

## 主な機能

- コマンドの一覧を表示する
- 入力に応じて先頭一致・部分一致で表示コマンドを絞り込む
- 先頭一致時に入力を補完する
- 常駐してホットキーにより起動する

## 操作方法

| キー          | 動作                                                               |
| ------------- | ------------------------------------------------------------------ |
| `Enter`       | 入力欄の右側のボタンと同じ動作（実行 or 検索）                     |
| `Esc`         | その右のボタンと同じ動作（閉じる等）                               |
| `Ctrl+Enter`  | 実行できるコマンドがある状態で、そのコマンドの設定ダイアログを開く |
| `Shift+Enter` | そのコマンドのリンク先のフォルダを開く                             |

## 「送る」へのショートカット登録

設定画面の「送るへコマンド登録用ショートカットを作成する」ボタンを押すと、
Windowsの「送る」メニューにショートカットが登録される。
以後、エクスプローラーでファイルを右クリックして「送る」→「らんちゃに登録」を選ぶと、
そのファイルをコマンドとして直接登録できる。
