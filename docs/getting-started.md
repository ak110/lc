# はじめに

## インストール

[Releases](https://github.com/ak110/lc/releases)からZIPをダウンロードし、任意のフォルダに展開してください。

## 基本的な使い方

タスクトレイに常駐し、設定したホットキーでランチャーを起動できます。
初回起動時にコマンドを登録してください。

## コマンドライン引数

| 引数       | 動作                                                                     |
| ---------- | ------------------------------------------------------------------------ |
| `/close`   | 既に常駐している場合、終了する                                           |
| `/restart` | 既に常駐している場合、再起動する                                         |
| それ以外   | ファイル名と見なし、コマンドを登録する（送るの「らんちゃに登録」の動作） |
