# らんちゃ

[![CI](https://github.com/aki-/lc/actions/workflows/ci.yml/badge.svg)](https://github.com/aki-/lc/actions/workflows/ci.yml)

コマンドラインランチャー。

## 主な機能

- コマンドの一覧表示（アイコン付き）
- 入力に応じて先頭一致・部分一致で表示コマンドの絞り込み
- 先頭一致時の入力補完
- 常駐してホットキーによる起動（キーは変更可能）
- 「送る」へコマンドの登録用ショートカットの作成

## インストール

Releaseページからダウンロードして、任意のフォルダに展開する。

## アンインストール

ショートカットを作成した場合は削除し、インストールしたファイルや使用時に作成されたファイルなどを削除する。

## 操作方法

- `Enter`キーは入力欄の右側のボタン、`Esc`キーはその右のボタンと同じ動作
- 実行できるコマンドがある状態（入力欄右側のボタンが「実行」になっている状態）で`Ctrl+Enter`を押すと、そのコマンドの設定ダイアログを開く
- `Shift+Enter`で、そのコマンドのリンク先のフォルダを開く

## コマンドライン引数

- `/close` — 既に常駐している場合、終了する
- `/restart` — 既に常駐している場合、再起動する
- それ以外 — ファイル名と見なし、コマンドを登録する（送るの「らんちゃに登録」の動作）
