# らんちゃ

[![CI](https://github.com/ak110/lc/actions/workflows/ci.yml/badge.svg)](https://github.com/ak110/lc/actions/workflows/ci.yml)
[![Release](https://github.com/ak110/lc/actions/workflows/release.yml/badge.svg)](https://github.com/ak110/lc/actions/workflows/release.yml)

コマンドラインランチャー＆ボタン型ランチャー。

## 主な機能

### コマンド型ランチャー

- コマンドの一覧表示
- 入力に応じて先頭一致・部分一致で表示コマンドの絞り込み
- 先頭一致時の入力補完
- 常駐してホットキーによる起動
- 「送る」へコマンドの登録用ショートカットの作成

### ボタン型ランチャー

- よく使うコマンドをボタンとしてグリッド状に配置
- 複数タブ対応
- ドラッグ&ドロップでボタンの並び替えやタブ間移動が可能
- ロック機能でフォーカスを外しても閉じないよう設定可能

## インストール

[Releaseページ](https://github.com/ak110/lc/releases)からダウンロードして、任意のフォルダに展開してください。

## アンインストール

ショートカットを作成した場合は削除し、インストールしたファイルや使用時に作成されたファイルなどを削除してください。

## 操作方法

### コマンド型ランチャー

- `Enter`キーは入力欄の右側のボタン、`Esc`キーはその右のボタンと同じ動作
- 実行できるコマンドがある状態（入力欄右側のボタンが「実行」になっている状態）で`Ctrl+Enter`を押すと、そのコマンドの設定ダイアログを開く
- `Shift+Enter`で、そのコマンドのリンク先のフォルダを開く

### ボタン型ランチャー

- ボタンをクリックでコマンドを実行
- ボタンを右クリックで編集・削除などのメニューを表示
- 余白やタブを右クリックで設定・終了などのメニューを表示
- ドラッグ&ドロップでボタンの配置変更やタブ間移動が可能

## コマンドライン引数

- `/close` — 既に常駐している場合、終了する
- `/restart` — 既に常駐している場合、再起動する
- それ以外 — ファイル名と見なし、コマンドを登録する（送るの「らんちゃに登録」の動作）
