# らんちゃ 取り扱い説明書

## はじめに

定番っぽい機能を大体搭載したつもりなコマンドラインランチャーです。
主な機能は以下の通りです。

- コマンドの一覧表示（アイコン付き）
- 入力に応じて先頭一致・部分一致で表示コマンドの絞り込み
- 先頭一致時の入力補完
- 常駐してホットキーによる起動（キーは変更可能）
- 「送る」へコマンドの登録用ショートカットの作成

## インストール

解凍して、適当なフォルダへ置いて、適当に起動してください。

## アンインストール

ショートカットを作成した場合は削除し、インストールしたファイルや使用時に作成されたファイルなどを削除してください。

## 操作方法

大体見た目通り動くと思います。

- `Enter` キーは入力欄の右側のボタン、`Esc` キーはその右のボタンと同じ動作になります。
- 実行出来るコマンドがある状態（入力欄右側のボタンが「実行」になってる状態）で `Ctrl+Enter` を押すと、そのコマンドの設定ダイアログを開きます。
- `Shift+Enter` で、そのコマンドのリンク先のフォルダを開きます。

## コマンドライン引数

- `/close` を付けて起動すると、既に常駐してる場合、終了します。
- `/restart` を付けて起動すると、既に常駐してる場合、再起動します。
- それ以外の場合はファイル名と見なし、コマンド登録を行います。（送るの「らんちゃに登録」の動作）
